1 前言

最常用的即BeanUtils#copyProperties(Object source, Object target)方法，优雅的拷贝bean属性。

source和target属性必须要相同类型，相同属性名

3 ObjectUtils

一些杂项的使用方法，针对实例对象，这个工具主要针对框架内部使用，浏览了一下方法发现蛮有用的。所属spring-core模块

3.1 判断对象是否为空

boolean ObjectUtils#isEmpty(java.lang.Object)

我们看下这个类的实现：

可以看到这个空判断覆盖了很多类型，包括String、Collection、Array，可以合并掉StringUtils、和CollectionUtils的isEmpty判空操作了

3.2 比较对象是否相等

比较对象是相等，如果都为null返回true。同时支持比较两个数组的每个元素是否相等，jdk自带的Objects#equals不支持比较数组是否相同(转而比较数组引用了)

ObjectUtils#nullSafeEquals(Object o1, Object o2)

3.3 安全的转成字符串

ObjectUtils#nullSafeToString(Object[] obj)

可以理解为增强型的toString方法，如果对象为null则转成”null”字符串，如果对象是数组类型，则转成数组形式的字符串，如果是普通对象，其实就是调用toString方法
