1 Algorithm 703. Kth Largest Element in a Stream 数据流中第K大元素

Design a class to find the kth largest element in a stream. Note that it is the kth largest element in the sorted order, not the kth distinct element.

Your KthLargest class will have a constructor which accepts an integer k and an integer array nums, which contains initial elements from the stream. For each call to the method KthLargest.add, return the element representing the kth largest element in the stream.

2 Review JVM, JDK, JRE: What’s the difference?

原文链接： https://www.javaworld.com/article/3347596/jvm-jdk-jre-whats-the-difference.html

这篇文章详细解释了JDK、JRE、JVM三者的关系和各自的用途，算是一篇基础拾遗文章吧，对于理清三者关系很有帮助，文章不仅详细解释了三者的区别，还介绍了三者各自的用途

简洁一点总结起来就是，摘自原文：

  • The JVM is the Java platform component that executes your programs.
  • The JRE creates the JVM and ensures dependencies are available to your programs.
  • The JDK allows you to create Java programs that can be executed and run by the JVM and JRE.

另外还可以参考一下Oracle的JDK组成图，很清晰 https://docs.oracle.com/javase/8/docs/index.html
